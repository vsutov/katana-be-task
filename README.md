
#  Katana Backend Programming Test

Dockerized REST API simulation of a card deck

- Create a new Deck:
	- `POST /deck`
- Open a Deck
- Draw a Card


## Technologies used
- Node 18 + Express 4
- Typescript 5
- Redis
- Jest + Supertest
- Yup

## Docker
### Build
`docker-compose build`

### Run
`docker-compose run app`

### Tests
`docker-compose run tests`
